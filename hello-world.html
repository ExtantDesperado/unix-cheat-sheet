Unix Cheat Sheet

Basic Unix commands and their uses

Unix Shell Command Name Description Example
ls List the files in the current directory

extantdesperado:~/workspace $ ls
README.md hello-world.html
cd Change the current directory to otherdir

extantdesperado:~ $ cd workspace/
extantdesperado:~/workspace $
pwd Print the current directory

extantdesperado:~/workspace $ pwd
/home/ubuntu/workspace

mkdir Make a new directory with the given name

extantdesperado:~ $ mkdir stuff
extantdesperado:~ $ ls
lib/ stuff/ workspace/

rmdir Remove the directory with the given name

extantdesperado:~ $ rmdir stuff
extantdesperado:~ $ ls
lib/ workspace/
